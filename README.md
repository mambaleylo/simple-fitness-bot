# 🏋️ Fitness Bot

Telegram-бот для домашних тренировок с подпиской.

## Возможности

- **Закреплённые тренировки** — разминка, заминка и базовые материалы, всегда доступны бесплатно
- **Тренировки месяца** — новые тренировки каждую неделю по подписке, хранятся 30 дней
- **Прогресс** — отметка выполненных тренировок
- **Уведомления** — автоматические уведомления подписчикам по расписанию
- **Автоочистка** — 1-го числа каждого месяца старые тренировки удаляются автоматически

## Установка

```bash
git clone https://github.com/mambaleylo/simple-fitness-bot
cd simple-fitness-bot
pip install -r requirements.txt
```

## Настройка

Открой `config.py` и замени:

```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "ВАШ_ТОКЕН_ОТ_BOTFATHER")
ADMIN_IDS = [123456789]  # ваш Telegram ID (узнать: @userinfobot)
```

## Наполнение демо-контентом

```bash
python seed_data.py
```

## Запуск

```bash
python bot.py
```

## Команды

### Для пользователей
- `/start` — главное меню

### Для администраторов
- `/admin` — панель администратора (кнопками)

Через панель доступно:
- Добавить закреплённую тренировку
- Добавить тренировку месяца
- Активировать подписку пользователю (30/60/90 дней)
- Список всех пользователей
- Рассылка всем пользователям
- Ручная очистка старых тренировок

## Расписание

Настраивается в `config.py`:

```python
SCHEDULE_DAYS = [0, 2, 4]  # ПН, СР, ПТ
SCHEDULE_TIME = "10:00"    # в 10:00 по Москве
```

По этому расписанию бот отправляет уведомления подписчикам о новых тренировках.
